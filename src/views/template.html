题库 JSON 模板

按照以下 JSON 格式准备题库文件，即可导入 PaperLens 进行刷题。每道题包含题目、四个选项、正确答案索引及解析。

或复制下方 JSON 自行粘贴
JSON 
[
  {
    "id": 1,
    "question": "以下哪个是 Python 中的可变数据类型？",
    "options": [
      "A. int",
      "B. str",
      "C. list",
      "D. tuple"
    ],
    "answer": 2,
    "explanation": "Python 中 list（列表）是可变的，可以修改其元素。int、str、tuple 都是不可变类型。"
  },
  {
    "id": 2,
    "question": "TCP 三次握手中，第三次握手的作用是什么？",
    "options": [
      "A. 确认双方收发能力正常",
      "B. 发送数据",
      "C. 关闭连接",
      "D. 重新传输"
    ],
    "answer": 0,
    "explanation": "第三次握手（客户端发送 ACK）确认了服务端的发送能力和客户端的接收能力均正常，完成双向验证。"
  }
]

可包含多个题目对象，数组长度即题库总数

字段说明

字段 类型 必填 说明
id number 是 题目唯一编号，不可重复
question string 是 题目内容，支持纯文本
options string[] 是 4 个选项的字符串数组，建议以 A. B. C. D. 开头
answer number 是 正确答案的索引 (0=A, 1=B, 2=C, 3=D)
explanation string 否 答案解析，答错后展示。可为空字符串

提示

PaperLens 也支持直接导入 Word (.docx)、Markdown (.md)、PDF 文件，系统会尝试自动解析其中的选择题。 但推荐使用 JSON 格式以确保解析准确性。
