Quiz JSON Template

Prepare quiz files in the following JSON format. Each question includes the question text, four options, correct answer index, and explanation.

Or copy the JSON below
JSON 
[
  {
    "id": 1,
    "question": "以下哪个是 Python 中的可变数据类型？",
    "options": [
      "A. int",
      "B. str",
      "C. list",
      "D. tuple"
    ],
    "answer": 2,
    "explanation": "Python 中 list（列表）是可变的，可以修改其元素。int、str、tuple 都是不可变类型。"
  },
  {
    "id": 2,
    "question": "TCP 三次握手中，第三次握手的作用是什么？",
    "options": [
      "A. 确认双方收发能力正常",
      "B. 发送数据",
      "C. 关闭连接",
      "D. 重新传输"
    ],
    "answer": 0,
    "explanation": "第三次握手（客户端发送 ACK）确认了服务端的发送能力和客户端的接收能力均正常，完成双向验证。"
  }
]

Multiple question objects allowed — array length = total questions

Field Descriptions

Field Type Required Description
id number 是 Unique question ID, no duplicates
question string 是 Question text, plain text
options string[] 是 Array of 4 option strings, suggest A. B. C. D. prefix
answer number 是 Correct answer index (0=A, 1=B, 2=C, 3=D)
explanation string 否 Answer explanation shown after wrong answer. Can be empty string.

Note

LearningTool Word (.docx)、Markdown (.md)、PDF 文件，LearningTool supports direct import of Word (.docx), Markdown (.md), and PDF files — auto-parsing is attempted. JSON is recommended for accuracy.。 。
